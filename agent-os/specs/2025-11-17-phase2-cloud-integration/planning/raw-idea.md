# Feature: PHASE 2 - Cloud Integration (Gemini Live API)

**Size:** M (Medium)
**Phase:** 2 (requires Phase 1 completion)

## Description

Add Google Gemini Live API (2.5 Flash) for cloud-based speech-to-text and text-to-speech processing. Implement hybrid switching logic based on connectivity with fallback to local STT/TTS. Ensure status indicator reflects processing state.

**Phase 1 Prerequisite:** Requires working local PoC with Temi SDK, local LLM (Gemma), and hardcoded menu/knowledge base files.
